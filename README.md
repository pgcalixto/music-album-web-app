# music-album-web-app
Web application using Node, React and MySQL to store information about music albums.

## Requirements

* MySQL
* npm

## Prepare

To prepare the environment and initialize data, run

```bash
./setup.sh
```

## Execute

To setup and run the server, run

```bash
npm start
```
